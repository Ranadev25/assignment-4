## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Answer::
1. getElementById = selects only one element by its ID.
2. getElementsByClassName = selects all elements with the same class.
3. querySelector = selects the first element that matches a CSS selector.
4. querySelectorAll = selects all elements that match a CSS selector (returns a static NodeList).


### 2. How do you create and insert a new element into the DOM?
Answer::
-First, create a new element. Then, add text, attributes, or classes to it. Finally, insert it into a parent element of the body.


### 3. What is Event Bubbling? And how does it work?
Answer::
-An event first occurs on the target element.Then it bubbles up through parent → grandparent → document.


### 4. What is Event Delegation in JavaScript? Why is it useful?
Answer::
-Attach a single listener on a parent element. The parent handles events triggered by its child elements.


### 5. What is the difference between preventDefault() and stopPropagation() methods?
Answer::
1. preventDefault() = prevents the default action of the event (e.g., link navigation or form submission).
2. stopPropagation() = stops the event from bubbling to parent elements.
